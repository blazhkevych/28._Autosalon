Стандартная библиотека С++. Контейнеры. Итераторы. Алгоритмы. Предикаты. Функторы.
	Создать приложение для работы автосалона. 
	Необходимо хранить информацию о продаваемых автомобилях 
(название, год выпуска, объем двигателя, цена). 
	Реализовать интерфейс для 
	* добавления, 
	* удаления, 
	* редактирования и 
	* печати данных. 
	Предусмотреть 
	* сортировку данных по различным параметрам, а также 
	* поиск данных по различным критериям.
При реализации приложения необходимо использовать контейнеры, итераторы, функторы, предикаты и алгоритмы
